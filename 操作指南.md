## 配置搜索引擎

### 支持的搜索引擎

| 搜索引擎 | 是否需要 API Key | 获取地址 |
|----------|------------------|----------|
| DuckDuckGo | ❌ 不需要 | 默认可用 |
| Bing | ✅ 需要 | https://www.microsoft.com/en-us/bing/apis/bing-web-search-api |
| Google | ✅ 需要 | https://developers.google.com/custom-search/v1/overview |
| Brave | ✅ 需要 | https://brave.com/search/api/ |
| SerpApi | ✅ 需要 | https://serpapi.com/ |
| SearXNG | ✅ 需要自建 | https://docs.searxng.org/ |

### 配置方法

#### 方法一：使用 .env 文件（推荐）

```bash
# 复制配置模板
cp .env.example .env

# 编辑配置文件
nano .env
```

在 `.env` 文件中填入你的 API Key：

```env
# Bing 搜索 API
BING_API_KEY=你的Bing密钥

# Google 自定义搜索 API
GOOGLE_API_KEY=你的Google密钥
GOOGLE_SEARCH_ENGINE_ID=你的搜索引擎ID

# Brave 搜索 API
BRAVE_API_KEY=你的Brave密钥

# SerpApi
SERPAPI_API_KEY=你的SerpApi密钥

# SearXNG（自建实例）
SEARXNG_URL=https://你的SearXNG地址
```

#### 方法二：使用环境变量

```bash
export BING_API_KEY=你的Bing密钥
export GOOGLE_API_KEY=你的Google密钥
export GOOGLE_SEARCH_ENGINE_ID=你的搜索引擎ID
```

### API Key 获取指南

#### Bing API Key

1. 访问 https://portal.azure.com
2. 创建 "Bing Search v7" 资源
3. 在 "Keys and Endpoint" 中获取 API Key

#### Google Custom Search API

1. 访问 https://console.cloud.google.com
2. 启用 "Custom Search API"
3. 创建 API Key
4. 访问 https://programmablesearchengine.google.com 创建搜索引擎
5. 获取搜索引擎 ID

#### Brave Search API

1. 访问 https://brave.com/search/api/
2. 注册并选择套餐
3. 获取 API Key

## 客户端配置

### Claude Desktop 配置

#### macOS

编辑文件：`~/Library/Application Support/Claude/claude_desktop_config.json`

```json
{
  "mcpServers": {
    "web-search": {
      "command": "node",
      "args": ["/你的完整路径/WebSearchApi/dist/index.js"],
      "env": {
        "BING_API_KEY": "你的Bing密钥",
        "GOOGLE_API_KEY": "你的Google密钥",
        "GOOGLE_SEARCH_ENGINE_ID": "你的搜索引擎ID"
      }
    }
  }
}
```

#### Windows

编辑文件：`%APPDATA%\Claude\claude_desktop_config.json`

### Cursor 配置

在项目根目录创建 `.cursor/mcp.json`：

```json
{
  "mcpServers": {
    "web-search": {
      "command": "node",
      "args": ["/你的完整路径/WebSearchApi/dist/index.js"],
      "env": {}
    }
  }
}
```

### 重启客户端

配置完成后，重启 Claude Desktop 或 Cursor 使配置生效。

## 工具使用示例

### 示例 1：基础网页搜索

**请求：**
```
请搜索"2024年人工智能最新进展"
```

**LLM 调用：**
```json
{
  "tool": "web_search",
  "arguments": {
    "query": "2024年人工智能最新进展",
    "max_results": 5
  }
}
```

### 示例 2：新闻搜索

**请求：**
```
搜索最近一周关于 ChatGPT 的新闻
```

**LLM 调用：**
```json
{
  "tool": "web_search",
  "arguments": {
    "query": "ChatGPT",
    "topic": "news",
    "time_range": "week",
    "max_results": 10
  }
}
```

### 示例 3：提取网页内容

**请求：**
```
帮我提取这篇文章的内容：https://example.com/article
```

**LLM 调用：**
```json
{
  "tool": "web_extract",
  "arguments": {
    "urls": "https://example.com/article",
    "format": "markdown"
  }
}
```

### 示例 4：爬取网站

**请求：**
```
爬取 https://docs.example.com 网站的所有 API 文档
```

**LLM 调用：**
```json
{
  "tool": "web_crawl",
  "arguments": {
    "url": "https://docs.example.com",
    "instructions": "查找所有 API 文档页面",
    "select_paths": ["/api", "/docs"],
    "max_depth": 3
  }
}
```

### 示例 5：生成网站地图

**请求：**
```
帮我分析 https://example.com 的网站结构
```

**LLM 调用：**
```json
{
  "tool": "web_map",
  "arguments": {
    "url": "https://example.com",
    "max_depth": 3,
    "limit": 100
  }
}
```

### 示例 6：深度研究

**请求：**
```
请研究一下人工智能对医疗行业的影响
```

**LLM 调用：**
```json
{
  "tool": "web_research",
  "arguments": {
    "query": "人工智能对医疗行业的影响",
    "max_sources": 15,
    "output_format": "report",
    "time_range": "year"
  }
}
```

### 示例 7：语义搜索

**请求：**
```
用语义搜索找关于量子计算的最新突破
```

**LLM 调用：**
```json
{
  "tool": "web_search",
  "arguments": {
    "query": "量子计算最新突破",
    "semantic": true,
    "max_results": 10
  }
}
```

## 常见问题

### Q1: 启动时报错 "Cannot find module"

**解决方案：**
```bash
# 重新安装依赖
rm -rf node_modules
npm install

# 重新构建
npm run build
```

### Q2: 搜索没有结果

**可能原因：**
1. 未配置任何搜索引擎 API Key
2. API Key 无效或已过期
3. 网络连接问题

**解决方案：**
1. 检查 `.env` 文件配置
2. 验证 API Key 是否有效
3. 检查网络连接

### Q3: Claude Desktop 无法连接 MCP 服务器

**解决方案：**
1. 检查配置文件路径是否正确
2. 确保 `dist/index.js` 文件存在
3. 重启 Claude Desktop
4. 检查 Node.js 路径：`which node`

### Q4: 搜索速度慢

**解决方案：**
1. 使用 `fast` 或 `ultra-fast` 搜索深度
2. 减少 `max_results` 数量
3. 配置更快的搜索引擎（如 Bing）

### Q5: 如何同时使用多个搜索引擎？

MCP 服务器会自动使用所有已配置的搜索引擎。你也可以在请求中指定：

```json
{
  "tool": "web_search",
  "arguments": {
    "query": "你的查询",
    "engine": "bing"
  }
}
```

### Q6: 如何更新到最新版本？

```bash
git pull
npm install
npm run build
# 重启服务
```
